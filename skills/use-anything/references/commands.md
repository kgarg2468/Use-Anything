# Command Reference

## Targets

- Package name: `use-anything requests`
- Docs URL: `use-anything https://docs.python-requests.org/en/latest/`
- GitHub repo URL: `use-anything https://github.com/pallets/flask`
- Local source directory: `use-anything /absolute/path/to/project`
- Binary: `use-anything --binary ffmpeg`

## Useful flags

- `--model codex-cli`
- `--probe-only`
- `--force`
- `--output-dir <path>`

## Validation

- `use-anything validate ./use-anything-requests`
